# Guide Étudiant

## Création de compte

1. Accédez à la page d'inscription
2. Choisissez "Étudiant"
3. Remplissez le formulaire :
   - Email étudiant
   - Mot de passe (minimum 8 caractères)
   - Prénom et nom
   - Établissement scolaire
4. Validez votre email

## Connexion

1. Cliquez sur "Connexion"
2. Entrez votre email et mot de passe
3. Vous êtes redirigé vers votre tableau de bord

## Trouver une entreprise

1. Parcourez la liste des entreprises partenaires
2. Utilisez les filtres :
   - Ville
   - Secteur d'activité
3. Cliquez sur une entreprise pour voir les détails

## Réserver un créneau

### Étape 1 : Choisir l'entreprise

Cliquez sur "Réserver" sur la fiche entreprise

### Étape 2 : Sélectionner la date et l'heure

1. Consultez le calendrier des disponibilités
2. Les créneaux verts sont disponibles
3. Les créneaux gris sont déjà réservés
4. Cliquez sur un créneau disponible

### Étape 3 : Confirmer

1. Ajoutez des notes pour l'entreprise :
   - Votre motivation
   - Type de stage/poste recherché
   - Questions spécifiques
2. Cliquez sur "Confirmer la réservation"
3. Vous recevez un email de confirmation

## Gérer vos réservations

### Voir vos réservations

Accédez à "Mes réservations" pour voir :

- **En attente** : L'entreprise n'a pas encore répondu
- **Confirmées** : Rendez-vous validé
- **Annulées** : Réservations annulées
- **Terminées** : Rendez-vous passés

### Modifier une réservation

!!! warning "Important"
    Vous ne pouvez modifier que les réservations "En attente"

1. Cliquez sur la réservation
2. Cliquez sur "Modifier"
3. Changez la date/heure ou les notes
4. Enregistrez

### Annuler une réservation

1. Cliquez sur la réservation
2. Cliquez sur "Annuler"
3. Indiquez la raison
4. Confirmez

!!! tip "Conseil"
    Prévenez au moins 24h à l'avance

## Notifications

Vous recevez des notifications pour :

- Confirmation de réservation
- Acceptation par l'entreprise
- Refus (avec raison)
- Rappel 24h avant le rendez-vous
- Annulation par l'entreprise

## Conseils

### Avant la réservation

- Consultez le site web de l'entreprise
- Préparez votre CV
- Rédigez des notes claires et professionnelles

### Pendant le rendez-vous

- Soyez ponctuel
- Tenue professionnelle
- Préparez vos questions
- Prenez des notes

### Après le rendez-vous

- Envoyez un email de remerciement
- Mettez à jour votre profil si besoin
